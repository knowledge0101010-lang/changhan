# 키워드 대응표 — 한(Han) 한글 ↔ 창제

UX 출처: 한 언어 키워드 문서 (로컬 `zodiac-lab/han` 참고)  
번역 목표: 창제 SDK 1.0.5  
차이 설명: [CANGJIE_VS_RUST.md](CANGJIE_VS_RUST.md)

## 제어

| 한글 | 창제 | 비고 (≠ 러스트/한) |
|------|------|-------------------|
| `함수` | `func` | 러스트 `fn` 아님 |
| `반환` | `return` | 창제는 마지막 식 암시 반환도 가능 |
| `변수` | `var` | 러스트 `let mut` 아님 |
| `상수` | `let` | 컴파일 시각 상수는 `const` (2차) |
| `만약` | `if` | 반드시 `(…)` |
| `이면` | _(삭제)_ | 창제에 then 키워드 없음 |
| `아니면` | `else` | |
| `그리고` | `&&` | |
| `또는` | `\|\|` | |
| `반복` | `for` | `for (x in …)` |
| `동안` | `while` | `while (…)` |
| `안에서` | `in` | |
| `멈춰` | `break` | 이름표(라벨) break 없음 |
| `계속` | `continue` | |

## 타입

| 한글 | 창제 |
|------|------|
| `정수` | `Int64` |
| `실수` | `Float64` |
| `문자열` | `String` |
| `불` | `Bool` |
| `없음` | `Unit`(반환) / `None`(Option) — 문맥에 따름 |

## 구조

| 한글 | 창제 | ≠ 러스트 |
|------|------|----------|
| `구조` | `struct` | 값 타입 |
| `클래스` | `class` | 한에 없는 창제 참조 타입 |
| `열거` | `enum` | `| 변형` 문법 |
| `맞춤` | `match` | `match (x) { case … => }` |
| `시도` | `try` | |
| `처리` | `catch` | |
| `포함` | `import` | 한의 파일 포함 ≠ 창제 패키지 import |
| `패키지` | `package` | |
| `확장` | `extend` | 러스트 `impl`에 가까움 |
| `인터페이스` | `interface` | |
| `상속` / 구현 기호 | `<:` | |

## 리터럴·내장

| 한글 | 창제 |
|------|------|
| `참` | `true` |
| `거짓` | `false` |
| `출력` | `println` |
| `출력오류` | 1차는 `println`으로 대체 (2차: 표준 오류) |
| `길이` | `.size` / 도우미 (2차) |

## 창제 2차 맵 (한에 없거나 다른 것)

| 한글(제안) | 창제 |
|------------|------|
| `생성` | `spawn` |
| `동기화` | `synchronized` |
| `속성` | `prop` |
| `공개` / `내부` / `비공개` | `public` / `internal` / `private` |
| `오버라이드` | `override` |
| `오픈` | `open` |
| `이즈` / `애즈` | `is` / `as` |

## 엔진 치환 순서

1. 문자열·주석 가리기  
2. 여러 토큰 구문 (`만약`…`이면`, `…동안`, `…안에서`)  
3. 키워드 최장 일치  
4. 가린 구간 복원  
5. 진입점 정규화 (`함수 메인` / `함수 main` → `main`)
