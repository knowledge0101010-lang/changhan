제품: **창한 (Changhan)** — 완전 한글 표면 → **Cangjie programming language (창제)** `.cj` 번역 엔진  
참고 UX: 한(Han)의 표면 완전 한글 (키워드·타입·내장·식별자·논리어)  
산술 기호(`+`, `>=`, `=`)는 한과 같이 **한글로 바꾸지 않음**

창제와 러스트·한의 차이: [spec/CANGJIE_VS_RUST.md](spec/CANGJIE_VS_RUST.md)

## 고정 규칙

- 완전 한글판 = **번역 엔진** (독립 LLVM 컴파일러 아님)
- **출시(SHIP)** = 제품 트렁크로 채택·배포. 1B는 학습만, 출시 금지
- 축을 섞어 설계하는 것은 SCORE 이전 금지 → 결과는 [SCORE.md](SCORE.md)
